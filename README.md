# SkyFocus
### Менеджер приложений и учёта времени

[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![C#](https://img.shields.io/badge/C%23-99.8%25-purple)](https://github.com/skysoftly/SkyFocus)

**SkyFocus** — это десктопное приложение для отслеживания времени, которое вы проводите за приложениями и играми. Оно помогает анализировать вашу цифровую активность, выявлять "пожирателей времени" и лучше планировать свой день.

## 🚀 Основные возможности

- **📊 Отслеживание в реальном времени**: Автоматический учет времени, проведенного в активных приложениях (браузеры, игры, офисные программы).
- **📈 Детальная статистика**:
  - Время за сегодня, неделю и месяц.
  - Графики активности по дням недели.
  - Сравнение с предыдущими периодами (например, "-64% меньше, чем в прошлом месяце").
- **🏆 Топ приложений**: Рейтинг самых используемых программ, чтобы вы знали, на что уходит ваше время.
- **⚙️ Управление приложениями**: Легко добавляйте или удаляйте приложения из списка отслеживания.
- **📁 Привязка к файлам**: Возможность отслеживать конкретные `.exe` файлы (например, `POWERPNT.EXE` для презентаций).
